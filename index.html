Sobre Este Sistema

Este es un sistema de gestión de blog conectado a una API REST desarrollada con Node.js, Express y MongoDB.

Características del sistema:

  • Crear, editar y eliminar artículos
  • Subir imágenes para los artículos
  • Búsqueda por ID de artículo
  • Interfaz responsiva y moderna
  • Conexión en tiempo real con la base de datos
